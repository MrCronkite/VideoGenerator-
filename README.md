# VideoGenerator


## Getting Started

1. Clone the repository.
2. Open:

```
AiVideoChat.xcodeproj
```

3. In Xcode select:

```
File → Packages → Resolve Package Versions
```

4. Wait until all Swift Package dependencies are downloaded.

5. Build and run the project (`⌘ + R`).

## Requirements

* Xcode 16+
* iOS 16+

## Notes

If package dependencies are not resolved automatically, manually run:

```
File → Packages → Resolve Package Versions
```

before building the project.
